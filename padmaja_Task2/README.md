# BMI Calculator 

## Overview
This project is a simple BMI (Body Mass Index) Calculator developed using Python as part of the Oasis Infobyte Internship Program (OIBSIP).  
It calculates BMI based on user input and classifies it into different health categories.

---

##  Features
- Accepts user input (weight in kg and height in meters)
- Calculates BMI using standard formula
- Displays BMI value with category:
  - Underweight
  - Normal weight
  - Overweight
  - Obese

---

##  Formula Used
BMI = weight / (height × height)
