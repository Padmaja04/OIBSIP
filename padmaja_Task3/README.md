# Random Password Generator 

##  Overview
This project is a Random Password Generator developed using Python as part of the Oasis Infobyte Internship Program (OIBSIP).  
It generates strong and secure passwords using a combination of letters, numbers, and symbols.

##  Features
- Generates random passwords
- Uses uppercase and lowercase letters
- Includes numbers and special characters
- User can specify password length

##  How It Works
The program uses Python's built-in libraries:
- 'random' for selecting random characters
- 'string' for accessing letters, digits, and symbols
