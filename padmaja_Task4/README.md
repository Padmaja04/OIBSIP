# Number Guessing Game 

##  Overview
This is a simple Python command-line game where the computer randomly selects a number between 1 and 100, and the user has to guess it correctly.

##  Features
- Random number generation
- User input based guessing
- Hints provided (Too high / Too low)
- Loop until correct answer

##  Concepts Used
- Python random module
- Loops (while loop)
- Conditional statements (if-elif-else)
- User input handling
